下面介绍一些学习网站

HTML: W3Cschool的html基础 可以先大概浏览一遍，知道各种标签的作用就够了。
CSS: 菜鸟教程的CSS 不必把每个特性都了解，有什么需求直接百度查解决方案就可以。
有兴趣可以了解一下bootstrap，一个前端UI框架，可以方便的布置封装好的UI。 bootstrap教程 和 bootstrap的官方文档
JavaScript: W3Cscool教程 了解基本的语法，会写函数调用即可。
拓展一下，VUE 是一个简单的前端框架。此次小学期没有要求，有兴趣的同学可以学习一下。

三者的关系(个人理解): HTML相当于骨架，CSS相当于外表，JavaScpirt相当于灵魂。

HTML 是一些基本的元素，像a标签等等，都是已有的标准规定好的东西，浏览器会对html文件进行解析，文件内显示什么标签，屏幕上就会显示什么元素。
CSS 就是对各种元素进行美化，将某个元素放到合适的位置，显示期望的样式。
JavaScript 则是负责和用户交互。如果没有JS，网页就单单是一张纸，只能看。合理的用JS进行开发，就可以在网页上和用户进行沟通。
